# Mark Drew's Blog

This is the personal blog of Mark Drew - a compendium of CFML and web technology topics.

Built with [Eleventy](https://www.11ty.dev/).

## Development

To run the site locally:

```bash
npm install
npm run serve
```

## Building

To build the site:

```bash
npm run build
```

The generated site will be in the `_site` directory.
