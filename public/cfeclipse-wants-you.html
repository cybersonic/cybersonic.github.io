CFEclipse wants you!

Posted by Mark Drew on code on March 11, 2006

Tagged under cfeclipse

A few people have been wanting to contribute to the CFEclipse project and that is awesome. So to get people started I thought I would point out a few resources and pages that will help.

Firstly it does help to join the contributors list.
This is where we discuss what is required and what people would like to add to the project.

Your next stop could be to read some of the bugs and see if you would like to be able to fix one of them. This would get you used to the code and we can help you get up to speed with the code.

Of course, the next stop would be to get the code itself, head over to the documents over at cfeclipse.org to let you know how to do this.

There is more information about what various bits are of the code do over at Spike’s CFEclipse Wiki which we are moving to the new CFEclipse.org Wikl (hey, that is ONE thing that needs to be done and needs people to help!)

So, you have a feature you have added now, or fixed. What do you do? Well, you can create a patch from Eclipse by right clickinng on the file you have changed and going to Team -> Create Patch. This creates a text file that you can then send to a contributor (Myself, Spike or Rob) and we can see if ti works.. test it , give whatever feedback and do a check in into the CVS repository.

I am currently looking for another couple of coders so that we can add a few more features and fix some bugs.

So now you know how it works… get in touch!
